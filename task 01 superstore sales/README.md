# Task 1:Superstore Sales Dashboard

##  Objective
Create an interactive Sales Dashboard using Microsoft Excel to analyze sales performance and identify key business insights. The dashboard is designed to provide a clear understanding of sales trends, product performance, regional sales distribution, and customer segments through Pivot Tables, Pivot Charts, and Excel visualizations.

##  Dataset
- Superstore Sales Dataset

##  Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Conditional Formatting
- Excel Functions

##  Dashboard Features
- Total Sales Analysis
- Sales by Category
- Sales by Sub-Category
- Sales by Region
- Sales by Segment
- Sales Trend Analysis
- Interactive Dashboard with Pivot Charts

##  Key Insights
- Analyzed overall sales performance across different regions and customer segments.
- Compared sales across product categories and sub-categories.
- Identified top-performing categories based on sales.
- Visualized sales trends to understand business performance over time.
- Built an interactive dashboard for easy sales analysis and decision-making.

##  Project Files
- `Task1_Sales_Dashboard.xlsx` – Excel Dashboard
- `README.md` – Project Documentation

##  Outcome
Successfully developed an interactive Sales Dashboard in Microsoft Excel that provides meaningful insights into sales performance. The dashboard enables users to analyze sales trends, compare category and regional performance, and make data-driven business decisions using Pivot Tables, Pivot Charts, and interactive visualizations.

> **Note:** The provided dataset did not include cost or expense information; therefore, profit analysis was not performed, and the dashboard focuses on sales-based insights.
