## Sales Dashboard(Task 01)

## Objective
Create an interactive Sales Dashboard using Microsoft Excel to analyze and visualize sales data. The dashboard provides key business insights using Pivot Tables, Pivot Charts, and Conditional Formatting.

## Dataset
- Superstore Sales Dataset

##  Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Conditional Formatting

## Dashboard Features
- Total Sales Overview
- Profit by Category
- Sales Trend Over Time
- Cleaned and Organized Dataset
- Interactive Charts
- Conditional Formatting to highlight high and low performers

##  Key Insights
- Calculated Total Sales from the dataset.
- Analyzed Profit by different product categories.
- Visualized monthly Sales Trends using a line chart.
- Applied Conditional Formatting to quickly identify high and low profit values.
- Built a user-friendly dashboard for business analysis.

## Project Files
- `Task1_Excel_Sales_Dashboard.xlsx` – Excel Dashboard
- `README.md` – Project Documentation

##  Outcome
Successfully developed an Excel Sales Dashboard that enables easy analysis of sales performance, category-wise profit, and sales trends using Pivot Tables and Charts.
